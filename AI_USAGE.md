# Engineering Decisions

## Bloc Architecture
Bloc was chosen for scalable state management and clean separation of concerns.

## Hive Database
Hive was selected because:
- Lightweight
- Fast local storage
- Offline support
- Easy Flutter integration

## OpenAI-Compatible API Design
The application supports configurable providers through:
- Base URL
- API Key
- Model Name

This allows compatibility with:
- OpenAI
- OpenRouter
- LM Studio

## UI/UX Improvements
Focus was placed on:
- Smooth animations
- Responsive layouts
- Minimal modern design
- Dark mode compatibility
- Improved user feedback
